# Simple Wallet Web App

Technical test written in Python 3.x and Django 2.x

## Requirements

Docker
Docker-compose

## Up & Run

```commandline
$ docker-compose build
$ docker-compose up
```

## Testing

```commandline
$ docker-compose run --rm --w /app/apzm  web ./manage.py test
```
